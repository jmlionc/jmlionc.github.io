üî∞ Introducci√≥n

En este m√≥dulo exploraremos las funciones de masa de probabilidad para variables discretas y las funciones de densidad de probabilidad (pdf) para variables continuas. Comprenderemos c√≥mo estas funciones nos permiten describir la probabilidad de que un evento ocurra en un conjunto de datos biol√≥gicos. Tambi√©n, profundizaremos en conceptos clave como el valor esperado y la varianza, que nos ayudan a caracterizar el centro y la dispersi√≥n de una distribuci√≥n.

Aprender√°s a utilizar el poder de R para trabajar con diferentes distribuciones de probabilidad. Cada distribuci√≥n en R tiene cuatro funciones clave, identificadas por un prefijo:

  • p: para la funci√≥n de distribuci√≥n acumulada (cdf) en un valor particular.

  • q: para los cuantiles (la funci√≥n inversa de la cdf) correspondientes a una probabilidad particular.

  • d: para la funci√≥n de densidad (pdf) en un valor particular.

  • r: para la generaci√≥n de valores aleatorios de una distribuci√≥n particular.

üéØ Objetivos

  1. Distinguir y aplicar las funciones de masa de probabilidad para variables discretas y las funciones de densidad para variables continuas.

  2. Calcular el valor esperado y la varianza para caracterizar distribuciones de probabilidad.

  3. Implementar en R los m√©todos de conteo como combinaciones, permutaciones y factoriales.

  4. Dominar las cuatro funciones clave (p, q, d, r) para trabajar con las principales distribuciones de probabilidad en R.

  5. Resolver problemas pr√°cticos de probabilidad, utilizando R como alternativa a las tablas estad√≠sticas para obtener valores de distribuci√≥n.

  6. Interpretar y generar datos pseudo-aleatorios a partir de distribuciones espec√≠ficas en R, comprendiendo la importancia de la replicabilidad.

üìñ Desarrollo

La probabilidad cl√°sica se basa en el principio de resultados igualmente probables. Si un experimento tiene un espacio muestral finito \(S\) con \(n\) resultados igualmente probables, y \(A\) es un evento de inter√©s dentro de \(S\), la probabilidad cl√°sica se define como: \[P(A)=\dfrac{n({A})}{n(S)}\] Donde:

  • \(P(A)\): probabilidad del evento

  • \(n(A)\): n√∫mero de resultados favorables al evento

  • \(n(S)\): n√∫mero total de resultados posibles en el espacio muestral \(S\).

En resumen, la probabilidad de un evento se obtiene dividiendo el n√∫mero de resultados favorables entre el total de resultados igualmente probables.

Lanzar una Moneda

Calculemos la probabilidad de obtener Cara (Heads) al lanzar una moneda justa.

outcomes <- c("Heads", "Tails")
total_outcomes <- length(outcomes)
favorable_outcomes <- length(outcomes[outcomes == "Heads"])

classical_prob <- favorable_outcomes / total_outcomes
classical_prob
[1] 0.5

Se define un experimento de lanzar una moneda:

  • outcomes <- c("Heads", "Tails"): crea un vector con los dos posibles resultados: ‚ÄúCara‚Äù y ‚ÄúCruz‚Äù.

  • total_outcomes <- length(outcomes): obtiene el n√∫mero total de resultados posibles, en este caso 2.

  • favorable_outcomes <- length(outcomes[outcomes == "Heads"]): cuenta los resultados favorables, es decir, cu√°ntas veces aparece ‚ÄúCara‚Äù en el vector (1 vez).

  • classical_prob <- favorable_outcomes / total_outcomes: calcula la probabilidad cl√°sica dividiendo los resultados favorables entre el total (1/2 = 0.5).

  • classical_prob: muestra la probabilidad de obtener ‚ÄúCara‚Äù al lanzar una moneda justa, que es 0.5.

Baraja de Cartas

Calcular la probabilidad de sacar una espada de una baraja est√°ndar de 52 cartas.

deck <- rep(c("Spades", "Hearts", "Diamonds", "Clubs"), each = 13)
total_cards <- length(deck)
spades <- length(deck[deck == "Spades"])

classical_prob_spade <- spades / total_cards
classical_prob_spade
[1] 0.25

Este c√≥digo nos ayuda a calcular la probabilidad de sacar una pica de una baraja normal. Como hay 52 cartas en total y 13 de ellas son picas, la probabilidad de sacar una pica es de 13 sobre 52, lo que se puede simplificar a 1 sobre 4 o 25 %. Por lo tanto, al sacar una carta de una baraja normal, hay un 25 % de probabilidad de que sea pica.

M√©todos de Conteo

Factorial

F√≥rmula

\[ n! = n \cdot (n-1) \cdot (n-2) \cdots 2 \cdot 1, \quad n \geq 0 \]

Propiedad especial \[0! = 1\]

Implementaci√≥n en R

factorial(n)

Permutaciones sin repetici√≥n

F√≥rmula

N√∫mero de formas de ordenar \(n\) elementos distintos:

\[P(n) = n!\]

N√∫mero de permutaciones de \(n\) elementos tomados de \(k\):

\[P(n,k) = \frac{n!}{(n-k)!}, \quad 0 \leq k \leq n\]

Implementaci√≥n en R

# Permutaciones de n elementos tomados de k
permutations <- function(n, k) {
  factorial(n) / factorial(n - k)
}

permutations(5, 3) # Ejemplo: permutaciones de 5 elementos tomados de 3
[1] 60

Permutaciones con repetici√≥n

F√≥rmula

Si hay \(n\) elementos con grupos repetidos de tama√±os \(n_1, n_2, \dots, n_r\):

\[P = \frac{n!}{n_1! \, n_2! \cdots n_r!}\]

Implementaci√≥n en R

# Permutaciones con repetici√≥n
permutations_with_repetition <- function(counts) {
  factorial(sum(counts)) / prod(factorial(counts))
}

# Ejemplo: letras de la palabra "MAMA"
permutations_with_repetition(c(2,1,1))
[1] 12

Combinaciones sin repetici√≥n

F√≥rmula

N√∫mero de formas de elegir \(k\) elementos de un conjunto de \(n\):

\[C(n,k) = \binom{n}{k} = \frac{n!}{k!(n-k)!}, \quad 0 \leq k \leq n \]

choose(n, k)

# Ejemplo: n√∫mero de formas de elegir 3 items de 5 items
choose(5, 3)

Combinaciones con repetici√≥n (combinaciones con reemplazo)

F√≥rmula

N√∫mero de formas de elegir \(k\) elementos de un conjunto de \(n\) con repetici√≥n:

\[ C_r(n,k) = \binom{n+k-1}{k} \]

Implementaci√≥n en R

  choose(n + k - 1, k)
}

[1] 6

Coeficiente Multinomial

F√≥rmula

Para particionar \(n\) elementos en \(k\) grupos de tama√±os \(n_1, n_2, \dots, n_k\):

\[\binom{n}{n_1, n_2, \dots, n_k} = \frac{n!}{n_1! \, n_2! \cdots n_k!}, \quad \sum_{i=1}^k n_i = n\]

multinomial_coefficient <- function(counts) {
  factorial(sum(counts)) / prod(factorial(counts))
}

# Ejemplo: particionar 5 en grupos de tama√±os 2,2,1
multinomial_coefficient(c(2,2,1))
[1] 30

Distribuciones Discretas

Distribuci√≥n Uniforme

Funci√≥n de Masa de Probabilidad

\[P(X = x) = \frac{1}{b-a+1}, \quad x = a, a+1, \dots, b\]

Valor Esperado

\[ E[X] = \frac{a+b}{2} \]

Varianza

\[ Var(X) = \frac{(b-a+1)^2 - 1}{12} \]

Distribuci√≥n Hipergeom√©trica

Funci√≥n de Masa de Probabilidad

\[ P(X = x) = \frac{\binom{K}{x} \binom{N-K}{n-x}}{\binom{N}{n}}, \quad x = 0,1,\dots,n \]

Valor Esperado

\[ E[X] = n \frac{K}{N} \]

Varianza

\[ Var(X) = n \frac{K}{N}\left(1-\frac{K}{N}\right)\frac{N-n}{N-1} \]

Distribuci√≥n Poisson

Valor Esperado

Aproximaciones

\(Hg(N,M,n)\longrightarrow Bin(n,p)\)

Sea \(0<p<1\). Si \(N\), \(M\longrightarrow\infty\) , de tal forma que \(\frac{M}{N} \longrightarrow p\). El Tama√±o de la poblaci√≥n \(N\) es ‚Äúgrande‚Äù en comparaci√≥n con el tama√±o de la muestra \(n\), es decir, \(\frac{n}{N}\longrightarrow 0\)

\(Bin(n,p) \longrightarrow Poi(\lambda)\) si \(0<p<1\) y \(n\cdot p \longrightarrow \lambda\) cuando \(n \longrightarrow \infty\) . Se tiene un \(n\) muy ‚Äúgrande‚Äù comparado con un \(p\) muy ‚Äúpeque√±o‚Äù

Uso de R como alternativa a las tablas de valores para las distribuciones.

La funci√≥n pnorm() - Probabilidad Acumulada - Calcula la probabilidad acumulada hasta un valor q (z), por defecto corresponde al √°rea a la izquierda de q.

pnorm(q = 1.96, mean = 0, sd = 1)
[1] 0.9750021

Devuelve ‚âà 0.975, que corresponde a la probabilidad de que una variable normal est√°ndar sea menor o igual que 1.96.

A este resultado es al que llegar√≠amos buscando en la t√≠pica tabla de los anexos de los libros de probabilidad. Usted puede comprobar el resultado observando la primera columna de la primera tabla dirigi√©ndose hasta el valor 1.9 y luego desplaz√°ndose de manera horizontal hasta hacer una intersecci√≥n con el valor .06 de la primera fila.

Uso de lower.tail

  • lower.tail = TRUE (valor por defecto): √°rea a la izquierda de q.
  • lower.tail = FALSE: √°rea a la derecha de q.
pnorm(q = 1.96, lower.tail = FALSE)
[1] 0.0249979

Devuelve ‚âà 0.025, que es el √°rea a la derecha de 1.96. Que tambi√©n podemos ver como 1 - 0.975.

Valor cr√≠tico (cuantil) - qnorm()

Hace el proceso inverso: dado un nivel de probabilidad acumulada p, devuelve el valor de q correspondiente en la distribuci√≥n.

[1] 1.959964

Devuelve ‚âà 1.96, el valor cr√≠tico de la normal est√°ndar para un nivel de confianza del 95%.

Ejemplo:

Supongamos que la estatura de los hombres en una poblaci√≥n se distribuye ~ Normal(Œº = 170, œÉ = 6).

  1. ¬øCu√°l es la probabilidad de que un hombre mida menos de 175 cm?

[1] 0.7976716

Es decir, 79.8%.

  1. ¬øCu√°l es la probabilidad de que mida m√°s de 175 cm?

[1] 0.2023284

Es decir, 20.2%.

Un hombre debe medir al menos 177.7 cm para estar en el 10% m√°s alto de la poblaci√≥n.
